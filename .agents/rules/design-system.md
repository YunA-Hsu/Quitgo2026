# 《社畜日記》Design System(鐵律,所有畫面必須遵守)

## 1. 色彩系統

### 七色情緒色票(唯一合法的情緒用色)
所有情緒相關 UI(日曆、能量罐、日記頁、情緒徽章、大富翁格子)只能使用以下七色:

| Token | 情緒 | 色碼 |
|---|---|---|
| `mood.happy` | 開心(向日葵黃) | `#FFD166` |
| `mood.calm` | 平靜(湖水藍) | `#7BC8D6` |
| `mood.ordinary` | 平凡的一天(鼠尾草綠) | `#B8C4A4` |
| `mood.achievement` | 成就感(琥珀金) | `#E6B655` |
| `mood.depressed` | 憂鬱(薰衣草灰紫) | `#9C9ABF` |
| `mood.hurt` | 委屈(霧藍灰) | `#A3B4C4` |
| `mood.angry` | 憤怒(珊瑚橙紅) | `#E76F51` |

### 禁令(最高優先)
- **禁止**在任何情緒相關元素使用紅色警告色,包含低能量、負面情緒、數值偏低的狀態。低能量用中性色或降低飽和度呈現,絕不能像「錯誤」或「危險」。
- **禁止** hardcode 色碼。所有顏色一律 import 自 `src/styles/tokens.ts`。
- 表單驗證錯誤等純功能性錯誤,可用柔和的 `#E76F51`(同珊瑚橙),但不得出現驚嘆號三角警告圖示搭配情緒內容。

### 基底色
- 全域背景:暖米色 `#F5F0E6`,疊加 10% 透明度格紋紙質感(`backgrounds/grid-paper.png` 平鋪)
- 主文字:暖深灰 `#4A4238`
- 次要文字:`#8C8478`
- 卡片底色:`#FDFBF5`,圓角 16px,陰影極淡(`0 2px 8px rgba(74,66,56,0.06)`)

## 2. 字體
- 只允許 `'Noto Sans TC', 'PingFang TC', sans-serif`
- **嚴禁**任何手寫風、書法風字體(裝置相容性問題)
- 層級:頁面標題 20px/600、區塊標題 17px/600、內文 15px/400、輔助 13px/400
- 行高一律 1.6 以上,中文閱讀舒適度優先

## 3. 語氣與情緒安全(產品靈魂)
- 核心原則:**陪伴,不評價**。所有文案與視覺回饋不得帶有評判、催促、警告意味。
- AI/系統回饋用觀察式語氣(「最近的壓力好像集中在⋯」),禁止命令式(「你應該⋯」)。
- 負面情緒紀錄後的回饋要溫暖有共鳴,不說教、不過度雞湯。
- 首頁與通知避免直接外顯「離職」等敏感字眼(隱私考量)。

## 4. 元件通則
- 手機優先:以 390px 寬度為基準,單欄佈局
- 觸控目標最小 44×44px
- 中文文案**集中**在 `src/copy/zh-TW.ts`,禁止寫死在元件裡
- 角色插圖(阿芽、八大居民)只能取用 `src/assets/` 現有素材,依 `ASSETS.md` 索引選用;**禁止**自行生成、外連或使用 placeholder 圖

### 情緒的視覺呈現(Mood Ball 規則)
- 情緒一律用心情球素材呈現(src/assets/icons/badge-*.png),不用純色塊
- 唯一例外:顯示尺寸小於 32px 時(如日曆格),改用對應七色的純色圓點
- 所有情緒顯示必須透過 `<MoodBall>` 元件,禁止直接 `<img>` 引素材
- 選中狀態:球微放大 1.08 + 柔和光暈(不可用外框線);未選:100% 不透明度,不做灰階

### 安全區域(Safe Area)規則
- index.html 的 viewport meta 必須含 viewport-fit=cover
- 底部固定元素(底部導覽列、固定 CTA 按鈕)必須加:
  `padding-bottom: calc(基礎padding + env(safe-area-inset-bottom))`
- 頁面可捲動內容的最底部,額外預留 `max(24px, env(safe-area-inset-bottom))` 的空間,避免最後一張卡片貼死在導覽列上
- 頂部瀏海同理:全螢幕背景元素延伸到頂,但內容從 `env(safe-area-inset-top)` 之下開始
- 開發預覽基準改為 393×852(iPhone 16),並用瀏覽器裝置模擬驗證

### 首頁日期條(Date Strip)規則
- 一次渲染 9 週（過去 8 週 + 本週），以週為單位（週一起始）
- 使用原生滾動 (`overflow-x: auto`)，搭配 `scroll-snap-type: x mandatory` 讓每週自動吸附；支援滑鼠拖曳與原生觸控滑動；初始定位在本週
- 日期顯示規則：
  - **已記錄**：日期數字放在 36px 實心情緒色圓圈上，文字顏色為深色 `#4A4238`（不使用圓點）
  - **未記錄**：只有數字，無背景色，不分過去未來皆正常顯示；禁止「漏記」等責備性文案
  - **今天**：
    - 若已記錄：情緒圓外圍加 2px 嫩芽綠 (`base.brandSprout`) 描邊
    - 若未記錄：36px 嫩芽綠空心描邊圓，文字加粗
- 點選行為：過去未記錄日 → 回傳 `canBackfill: true`；未來日 → 回傳 `canBackfill: false`；已記錄日 → 回傳 `recorded: true`
- 隨著滑動，上方需要顯示當前週所對應的年份與月份
- 離開今天所在的週時顯示「回到今天」按鈕
- 日期 key 一律 YYYY-MM-DD(Asia/Taipei)

### 能量罐卡片(Energy Jar)規則
- 今日金幣：每日歸零、上限 100，進度條用鼠尾草綠 (`base.brandSprout`)，滿檔不用紅色或警示色
- 累積金幣必須同時可見（小字「累積 N 枚」），擲骰扣的是累積金幣(30/次)
- 心情盤點列：七顆心情球圖檔 (`badge-*.png`) 40px 固定順序，下方為當日該情緒次數；
  0 次 = 球 60% 透明度 + 數字 0；≥1 次 = 全彩 + 數字加粗；禁止灰階或縮小
- 盤點列日期與 DateStrip 選取日連動

### 日記詳情規則
- 路由 /diary/:id,右滑入場;返回保留封存頁捲動位置與選中日期
- 日記送出後不可編輯(產品原則:保留當下真實紀錄);唯一補救為「收回」重寫
- 收回確認用溫和文案與次要色按鈕,禁止紅色刪除鈕
- /diary/new 送出鈕上方需顯示所選心情與標籤的視覺小結(送出前最後確認)
- 列表項整列可點,無內文的筆以事件標籤作為摘要

## 5. 大富翁地圖(職涯大富翁)專屬規則
- 平面 2D 插畫風,**不是**等角 3D
- 每次擲骰消耗 30 能量;骰子按鈕外圈為能量環
- 每一格都有幸運獎勵,落地才揭曉;稀有度三階(普通/稀有/史詩)用低飽和度框色區分,不用螢光色
- 未造訪格子使用迷霧(fog of war)效果
- 易經卜卦取代機會/命運卡
- 八大居民僅作為格子上的小型點綴,不得成為畫面主角
